# 🧠 WanderWise – AI Powered Smart Trip Planner  

![React](https://img.shields.io/badge/Frontend-React-blue)
![Firebase](https://img.shields.io/badge/Backend-Firebase-orange)
![Gemini](https://img.shields.io/badge/AI-Google%20Gemini-purple)
![Maps](https://img.shields.io/badge/API-Google%20Maps-green)

WanderWise is a full-stack AI-powered travel planning application that generates intelligent, personalized itineraries in seconds using **Google Gemini** and real-time location data from **Google Maps APIs**.

It eliminates manual trip research by automatically creating structured, day-wise travel plans with hotels, attractions, and optimized routes - all securely stored using Firebase.

> Plan smarter. Travel better. 🌍✨

---

## 🚀 Why WanderWise?

Planning trips manually takes hours of research across blogs, maps, and booking sites.

WanderWise simplifies this by:
- 🧠 Generating AI-based day-wise itineraries instantly  
- 🗺️ Showing real-time locations on interactive maps  
- 🔐 Securing user data with Firebase Authentication  
- ☁️ Saving trips in the cloud for anytime access  

---

## 🌟 Core Features

### 🔮 AI-Powered Travel Plans
- Uses Google Gemini API  
- Generates structured day-by-day itineraries  
- Tailored to destination and trip duration  

### 🗺️ Real-Time Maps & Places Integration
- Displays hotels, attractions, and routes  
- Uses Google Maps & Places APIs  
- Interactive map-based experience  

### 🔐 Secure Authentication
- Google Sign-In via Firebase Auth  
- User-specific saved itineraries  

### ☁️ Cloud Storage
- Real-time sync using Firebase Firestore  
- Persistent trip history  

### 📱 Responsive UI
- Built with React + TailwindCSS  
- Optimized for both desktop and mobile

## ⚙️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/tanikaaaa/WanderWise.git
cd WanderWise
```

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
npm run dev
```

Visit:

```
http://localhost:5173
```
