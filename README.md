# 🧠 WanderWise - AI Trip Planner

WanderWise is a full-stack web application that uses **Generative AI (Google Gemini)** and **Google Maps APIs** to help users plan smart, personalized travel itineraries in seconds. It suggests hotels, attractions, and routes tailored to the destination and trip duration, while securely storing user data with Firebase. With a clean, responsive interface and real-time map interactions, WanderWise makes trip planning faster, easier, and more enjoyable.

---

## 🌟 Key Highlights

- 🔮 **AI-powered Travel Plans** – Generates optimized, day-by-day itineraries based on your chosen location and trip length.
- 🗺️ **Live Map Integration** – Explore places, routes, hotels, and attractions directly on interactive Google Maps.
- 🔐 **Secure Login** – Google Sign-In via Firebase Authentication keeps your account and plans safe.
- ☁️ **Cloud Storage** – Plans are saved and synced in real time with Firebase Firestore.
- 💡 **Simple, Intuitive Design** – Built with React and TailwindCSS for a smooth user experience across devices.

---

## 🛠️ Tech Stack

### 📌 Frontend:
- [React.js](https://reactjs.org/) – UI framework
- [TailwindCSS](https://tailwindcss.com/) – Utility-first CSS styling

### 📌 Backend:
- [Node.js](https://nodejs.org/) – Server runtime

### 📌 APIs & Services:
- [Google Gemini API](https://ai.google.dev/) – AI-based itinerary generation
- [Google Maps & Places APIs](https://developers.google.com/maps/documentation) – Location data and map display
- [Firebase Auth](https://firebase.google.com/products/auth) – Secure authentication
- [Firebase Firestore](https://firebase.google.com/products/firestore) – Cloud NoSQL database
